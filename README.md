## 📁 Структура

| Папка | Описание |
|-------|----------|
| `api_tests/` | API-тесты на pytest + requests |
| `ui_tests/` | UI-тесты на Selenium + Page Object Model |
| `test_cases/` | Тест-кейсы и чек-листы в Markdown |
| `sql_tasks/` | SQL-запросы для практики QA |

## 🛠 Стек

- Python 3.11+
- pytest, requests
- Selenium WebDriver
- Git, Postman

## 🚀 Запуск

```bash
pip install -r requirements.txt

# API тесты
cd api_tests && pytest -v

# UI тесты
cd ui_tests && pytest -v
```
