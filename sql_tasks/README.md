# SQL-задачи — База данных университета ГУАП

Придумал учебную схему БД студентов ГУАП и написал запросы,  
которые QA-инженер использовал бы при тестировании учебного портала.

## Схема

```
students  ──< grades >── subjects
                              ^
schedule >─────────────── subjects
```

## Задачи

| # | Запрос | Зачем QA |
|---|--------|----------|
| 1 | Студенты группы Z3420 | Проверить зачисление |
| 2 | Количество студентов по группам | Сверить с деканатом после импорта |
| 3 | Средний балл за семестр | Сверить с UI |
| 4 | Студенты с двойками | Проверить уведомления о задолженности |
| 5 | Дублирующиеся email | Проверка уникальности |
| 6 | Расписание на понедельник | Сверить расписание в приложении с БД |
| 7 | Студенты без оценок | Целостность после начала сессии |
| 8 | Предметы без сданных экзаменов | Целостность данных |
| 9 | Топ-5 по среднему баллу | Проверить фичу «Рейтинг» |
| 10 | Экзамены по дням недели | Аналитика нагрузки |
| 11 | NULL в критических полях | Проверка после миграции |
| 12 | Удаление тестовых данных | Teardown после автотестов |
